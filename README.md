# Student Performance & Attendance Analytics

This project analyzes student academic performance using a dataset of 1,500+ records to uncover insights that could help educators and school administrators improve student outcomes.

This project is part of my *1 Project Per Week Data Analytics Challenge, and this is **Week 2*.

## Tools Used

- Microsoft Excel – Data cleaning with Power Query  
- PostgreSQL – Data analysis and SQL queries  
- Microsoft Power BI – Interactive dashboard creation

## Project Workflow

1. Data cleaning using Excel Power Query  
2. Data exploration and analysis using PostgreSQL  
3. Building an interactive dashboard in Power BI

## Key Business Questions

- Which subjects have the highest average scores?  
- Which student classes perform best or worst?  
- Does attendance influence student performance?  
- How do teachers compare in terms of student outcomes?  
- What is the distribution of student performance categories?

## Key Insights

- Average student score is *66.28*  
- Most students fall into the *Good performance category*  
- Academic performance is fairly consistent across subjects  
- Teacher-level differences exist in student outcomes  
- Performance differences across attendance categories are minimal
